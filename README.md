# 3D Interactive Portfolio — Aman Singh

An immersive, interactive 3D portfolio showcasing technical business analysis, full-stack development, and operations automation projects.

## 🚀 Live Demo
* Deployed Site: [itsaman4u.vercel.app](https://itsaman4u.vercel.app)

## 🛠️ Tech Stack
* **Frontend**: React, TypeScript, Tailwind CSS, shadcn/ui, Framer Motion
* **3D Rendering**: Three.js, React Three Fiber (R3F), React Three Drei
* **State & Routing**: React Router DOM, React Query
* **Deployment**: Vercel

## 🌟 Key Features
* **Interactive 3D Scene**: Floating interactive project blocks with orbital camera controls and particle effects.
* **Skills Dashboard**: Matrix of technical proficiency spanning backend development, databases, and operations automation.
* **Interactive Timeline**: Visual career journey from Operations Lead to Technical Business Analyst.
* **AI Chat Assistant**: Preconfigured AI agent to answer portfolio, resume, and experience-related questions.
